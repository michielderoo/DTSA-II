DTSA-II is a graphical user interface for the EPQ library. EPQ is a library of quantitative algorithms for electron probe microanalysis. EPQ was developed by the Microanalysis Group at the National Institute of Standards and Technology to provide the microanalysis community with a standardized set of quantification algorithms. It is a platform on which the basics have been implemented so as a community we can push the state-of-the-art in microanalysis forward. It is designed as a library so that it can be readily integrated into more sophisticated applications.
EPQ has been developed in the Java language for portability - it can run on many Linux, Unix (including Apple OSX) and Microsoft Windows computers. The source code is freely available to all. This is important. As a scientific community is it critical that we say precisely how our data is processed. While most researchers won't want to carefully study the quantitative algorithms within the EPQ library, they should feel confident that if there is ever a question they can go back to the source code and see precisely how the algorithm is implemented. EPQ is not a black box that asks to be trusted blindly.
